# Smart Self-Service Laundry System

An IoT-based self-service laundry model using STM32, ESP32, Blynk Cloud, Node.js, React, and payOS QR payment.

## Features

- QR payment with payOS
- Automatic washer/dryer control
- ESP32 ↔ STM32 UART communication
- Blynk IoT remote monitoring
- Manual admin mode
- React + Vite frontend
- Node.js + Express backend

## Tech Stack

- STM32
- ESP32
- Blynk IoT
- Node.js + Express
- React + Vite
- payOS
